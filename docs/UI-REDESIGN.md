# UI 전면 개편안 (AI Search Portal)

## 1) 목표와 톤
- 목표: “여러 AI 결과를 빠르게 비교하고, 신뢰도/근거를 한눈에 판단”하는 생산성 UI
- 톤: 실험실(Lab) + 분석대시보드(Analytics) + 저널(Journal) 느낌의 하이브리드
- 키워드: 투명성, 신뢰, 속도, 비교

## 2) 정보 구조 (IA)
1. **Dashboard**
2. **New Analysis**
3. **Compare**
4. **History**
5. **Exports (Notion/Markdown)**
6. **Settings**

좌측 고정 내비게이션 + 상단 작업바(현재 세션/실행 상태)

## 3) 레이아웃 개편안 (핵심 플로우)
### A. New Analysis (메인 화면)
- 상단: 분석 세션 제목/태그/목표(짧은 입력)
- 중앙: 프롬프트 입력 + “추가 질문/검증 라운드” 토글
- 하단: 에이전트 선택 칩(Perplexity/ChatGPT/Gemini/Claude) + 실행 옵션
- 우측 패널: 실행 큐 + 진행률 타임라인 (Socket 기반)

### B. Results (실행 후)
- 좌측: 타임라인(라운드별 로그)
- 중앙: 결과 카드 그리드(각 모델 결과)
- 우측: “합의/불일치” 요약 위젯 + 신뢰도 추정(휴리스틱)
- 하단 고정: “비교 보기” 토글 + “내보내기”

### C. Compare (심화 비교)
- 2~4분할 패널(가로/세로 스플릿)
- 결과 간 하이라이트 차이 표시(문장 단위)
- 공통 근거/참고 링크 추출 요약 박스

### D. History
- 시간순 카드형 리스트 + 필터(날짜, 모델, 태그)
- 카드 클릭 시 우측 슬라이드 인스펙터(결과 상세)

## 4) 디자인 시스템 (시안 요약)
### 컬러
- Base: Warm Gray + Ink Black
- Accent 1: Electric Teal (성공/활성)
- Accent 2: Amber (주의/경고)
- Accent 3: Coral (불일치/리스크)
- 배경: 미세 그라디언트 + 노이즈 텍스처

### 타이포그래피
- KR 본문: “IBM Plex Sans KR”
- 헤드라인/숫자: “Space Grotesk”
- 강조: 콘덴스 헤드라인으로 리듬감 제공

### 컴포넌트 룩
- 카드: 반투명 유리질감(글래스 아님) + 얇은 라인
- 버튼: pill 형태 + 얕은 음영 + 미세한 이동 모션
- 태그/칩: 상태별 컬러 라벨 + 아이콘
- 결과 텍스트: 문장 단위 하이라이트 레이어

## 5) 모션 가이드
- 페이지 로드: 120~180ms 페이드 + Y 이동
- 카드 등장: 스태거(한 줄씩)
- 비교 하이라이트: 250ms 강조 레이어 확산
- 타임라인 업데이트: 점등 애니메이션

## 6) 데이터 시각화
- **합의 지수**: 도넛 또는 게이지
- **불일치 히트맵**: 문장 단위 색상 스케일
- **응답 시간**: 미니 스파크라인

## 7) UX 세부
- “요약/원문/근거” 3탭 전환
- 결과 카드 상단에 “신뢰도/근거 수/응답 시간” 표시
- 비교 모드에서 모델명 고정 스티키
- 길이 긴 답변: 문단 접기/펼치기 + 빠른 목차

## 8) 모바일 대응
- 상단 탭 내비게이션으로 축소
- 결과 카드는 세로 스택
- 비교 모드: 스와이프 전환

## 9) 구현 제안 (현재 스택 기준)
- `client/`에 디자인 토큰(`styles/tokens.css`) 도입
- Tailwind 커스텀: 폰트/색상/그림자 확장
- 결과 카드/타임라인/비교 패널 컴포넌트 분리
- 스켈레톤 로딩 컴포넌트 추가

## 10) 시안 우선순위 (Phase)
1. **New Analysis + Results** (핵심 플로우)
2. Compare 화면
3. History 화면
4. Settings/Export 화면
